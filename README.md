# Seattle Sakura Mood Map 🌸

> **"不要查攻略了，让樱花来匹配你的情绪。"**

A poetic, emotion-driven lifestyle product built with an **Extreme Minimalist + Dreamy Frosted Glass** aesthetic. Designed for the 2026 Seattle Sakura season.

## 💎 Visual Philosophy: The Shinkai Aesthetic
Inspired by the high-key, soft-focus world of Makoto Shinkai, this application prioritizes breathtaking simplicity and fluid interaction.

- **Frosted Matte UI**: Heavy `45px` backdrop blur for a premium glassmorphic tactile feel.
- **Pearl Palette**: A curated mix of Pearl Pink (#F7E7E9), Fog White, and Icy Blue.
- **Cinematic Particles**: 3D falling sakura petals with a stylized 2px blur for depth-of-field.
- **Poetic Motion**: Slow, intentional 1.5s transitions and interactive 3D parallax cards.

## 🌸 Core Interaction Flow
1. **The Mood Selector**: Absolute minimalist landing with 4 curated emotional states.
2. **The Destination Carousel**: 3D stacked cards featuring verified full-bloom spots in Seattle/Eastside.
3. **Verified Assets**: All 8 destination images are manually verified for authenticity and loading stability.

## 🛠️ Technology Stack
- **Engine**: Vanilla JavaScript (ES6+).
- **Styling**: Modern CSS3 (Flexbox, Grid, 3D Transforms, Backdrop-filters).
- **Assets**: Verified High-Key Unsplash Assets.
- **Architecture**: Single-file execution (`index.html`) for maximum portability and speed.

## 🚀 Getting Started
Simply open `index.html` in any modern browser. No build steps or external dependencies required.
